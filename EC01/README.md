# EC01 - Landing Page SkillHub

## Description
Landing page publique de la plateforme SkillHub pour EC01 du Bachelor Développeur Web Full Stack.

## Structure du projet
```
EC01/
├── index.html       # Landing page principale
├── css/
│   └── style.css   # Styles avec Media Queries
├── js/
│   └── main.js     # JavaScript natif (ES6+)
└── package.json     # Configuration du projet
```

## Contraintes EC01
- ✅ Aucun Framework CSS/JS (Vanilla HTML/CSS/JS)
- ✅ Sémantique HTML5
- ✅ Accessibilité (ARIA, navigation clavier Tab/Focus)
- ✅ Responsive Design (Media Queries)
- ✅ Validé W3C

## Fonctionnalités
- Header avec navigation
- Section Hero avec boutons
- Section "Nos Valeurs" avec animations
- Section "Technologies"
- Footer avec contact
- Navigation fluide (smooth scroll)
- Animations CSS sur les cartes
